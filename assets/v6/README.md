# Assets V6

Ruta reservada para recursos visuales del runtime V6.

## Convención inicial

- `assets/v6/materials/`: materiales de bridge/render.
- `assets/v6/meshes/`: placeholders o mallas finales.
- `assets/v6/textures/`: texturas del pipeline 3D.

## Estado actual (Sprint 08)

El módulo `v6_render_bridge_3d` usa placeholders generados en runtime (cubos + material básico), por lo que esta carpeta queda documentada para reemplazo progresivo sin romper contratos.
